# aws-cloudformation-blueprints
Modular AWS CloudFormation (IaC) templates for Data Engineering and DevOps best practices.

A collection of modular AWS CloudFormation blueprints structured following Infrastructure as Code (IaC) best practices. Contains production-ready templates for common workloads such as web servers, data pipelines and scalable microservices. Focus is placed on separating resources by lifecycle (Network, Compute, Data) and the use of Nested Stacks for component management. Ideal for portfolio projects and rapid prototyping.
